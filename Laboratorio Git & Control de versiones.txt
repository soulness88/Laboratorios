Laboratorio Git & Control de versiones
1- Configurar un usuario git en local
git config --global user.name "soulness88"
git config --global credential.helper cache

2-Generar un repositorio en git y clonarlo a local por linea de comandos

git clone https://github.com/soulness88/Laboratorios.git
Username: soulness88

3-Subir el fichero de data.txt y el script

4-Generar una nueva rama en local, ejecutar el script y subir el resultado a github (se que es
ineficiente, revisad en remoto que la rama nueva exista, solo quiero eso)

5-Mergear la nueva rama a main y sincronizar con el repositorio remoto (comprobad que los cambios han subido)

*Ejercicio bonus. Guardar un token personal en este mismo fichero e intentar mergearlo en local y despues intentar syncronizarlo en github.
